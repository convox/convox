# Convox

Convox is an open-source [PaaS](https://en.wikipedia.org/wiki/Platform_as_a_service) based
on Kubernetes available for multiple cloud providers.

## Supported Clouds

- Amazon Web Services
- Digital Ocean
- Google Cloud
- Microsoft Azure
