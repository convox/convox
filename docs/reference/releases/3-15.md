# 3.15 Releases

Convox 3.15 updates the VPC CNI plugin to v1.14, upgrades BuildKit to v0.12.4, and adds rack name validation at install time.

## 3.15.0

**Released:** 2024-01-15

**Updates**

- Updated VPC CNI plugin to v1.14 ([PR #735](https://github.com/convox/convox/pull/735))
- Updated BuildKit to v0.12.4 ([PR #736](https://github.com/convox/convox/pull/736))
- Added ignore for BuildKit image cache errors ([PR #736](https://github.com/convox/convox/pull/736))

[View on GitHub](https://github.com/convox/convox/releases/tag/3.15.0)
