# 3.19 Releases

Convox 3.19 upgrades Kubernetes to 1.29 and introduces initContainer support, startup probes, app config management, and EFS volumeHandle support. This is also the release line that added DNS-01 certificate support for internal routers and user data script parameters.

## 3.19.0

**Released:** 2024-09-20

**Updates**

- Upgraded Kubernetes to v1.29 ([PR #820](https://github.com/convox/convox/pull/820))
- Upgraded Go to v1.23 ([PR #820](https://github.com/convox/convox/pull/820))
- Upgraded BuildKit to v0.16.0 ([PR #820](https://github.com/convox/convox/pull/820))
- Updated EBS CSI Driver to v1.34.0 ([PR #820](https://github.com/convox/convox/pull/820))
- Updated CoreDNS to v1.11.1 ([PR #820](https://github.com/convox/convox/pull/820))
- Updated EFS CSI Driver to v2.0.7 ([PR #820](https://github.com/convox/convox/pull/820))
- Updated VPC CNI to v1.18.3 ([PR #820](https://github.com/convox/convox/pull/820))
- Fixed rack creation and GitHub Action failures on GCP ([PR #820](https://github.com/convox/convox/pull/820))

[View on GitHub](https://github.com/convox/convox/releases/tag/3.19.0)
