# 3.23 Releases

Convox 3.23 upgrades Kubernetes to 1.33 and introduces VPA (Vertical Pod Autoscaler) and KEDA event-driven autoscaling. This release also adds Azure AKS node group and GPU support, along with significant component upgrades across all providers.

## 3.23.0

**Released:** 2025-11-05

**Feature Additions**

- Added `disableHostUsers` configuration for Linux containers ([PR #940](https://github.com/convox/convox/pull/940))
- Added GKE extended release channel support for managing minor version upgrades through Convox ([PR #940](https://github.com/convox/convox/pull/940))

**Updates**

- Upgraded Kubernetes to v1.33 ([PR #940](https://github.com/convox/convox/pull/940))
- Upgraded Golang to v1.24 ([PR #940](https://github.com/convox/convox/pull/940))
- Updated BuildKit to v0.25.1 ([PR #940](https://github.com/convox/convox/pull/940))
- Updated EBS CSI Driver to v1.51.1-eksbuild.1 ([PR #940](https://github.com/convox/convox/pull/940))
- Updated CoreDNS to v1.12.4-eksbuild.1 ([PR #940](https://github.com/convox/convox/pull/940))
- Updated EFS CSI Driver to v2.1.13-eksbuild.1 ([PR #940](https://github.com/convox/convox/pull/940))
- Updated Kube-proxy to v1.33.3-eksbuild.10 ([PR #940](https://github.com/convox/convox/pull/940))
- Updated VPC CNI to v1.20.4-eksbuild.1 ([PR #940](https://github.com/convox/convox/pull/940))
- Upgraded Azure module to v4.51 ([PR #940](https://github.com/convox/convox/pull/940))

**Fixes**

- Fixed Azure 4MB source upload issue by implementing chunked uploads using Put Range API ([PR #940](https://github.com/convox/convox/pull/940))

[View on GitHub](https://github.com/convox/convox/releases/tag/3.23.0)
