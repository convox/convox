# 3.17 Releases

Convox 3.17 upgrades Kubernetes to 1.28 along with updates to AWS VPC-CNI, kube-proxy, and the EBS driver.

## 3.17.0

**Released:** 2024-03-20

**Updates**

- Upgraded Kubernetes to v1.28 ([PR #756](https://github.com/convox/convox/pull/756))
- Updated AWS VPC-CNI to v1.16.4-eksbuild.2 ([PR #756](https://github.com/convox/convox/pull/756))
- Updated AWS kube-proxy to v1.28.6 ([PR #756](https://github.com/convox/convox/pull/756))
- Updated AWS EBS driver to v1.28.0 ([PR #756](https://github.com/convox/convox/pull/756))
- Updated telemetry ([PR #756](https://github.com/convox/convox/pull/756))

[View on GitHub](https://github.com/convox/convox/releases/tag/3.17.0)
