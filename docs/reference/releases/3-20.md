# 3.20 Releases

Convox 3.20 upgrades Kubernetes to 1.30 with major component updates across the stack, including BuildKit v0.19.0, Nginx Ingress Controller v1.12.0, and the migration to EKS-managed EBS CSI Driver v1.39.0.

## 3.20.0

**Released:** 2025-02-25

**Updates**

- Upgraded Kubernetes to v1.30
- Updated Docker BuildKit to v0.19.0
- Updated Cluster Autoscaler to v1.32.0
- Updated Metrics Server to v0.7.2
- Updated Nginx Ingress Controller to v1.12.0
- Updated EBS CSI Driver to EKS-managed v1.39.0
- Updated EFS CSI Driver to v2.1.4
- Updated CoreDNS to v1.11.4
- Updated Kube-Proxy to v1.30.9
- Updated VPC CNI to v1.19.2

[View on GitHub](https://github.com/convox/convox/releases/tag/3.20.0)

## 3.20.1

**Released:** 2025-03-12

**Feature Additions**

- Enhanced EKS node management with configurable additional node groups
- Added build pod node selection for optimized resource allocation
- Introduced custom AMI support for EKS clusters
- Added service placement controls for targeting custom node configurations

**Updates**

- Extended RDS provisioner with CreateServiceLinkedRole permission

**Fixes**

- Resolved error messages by disabling EBS snapshotter sidecar component

[View on GitHub](https://github.com/convox/convox/releases/tag/3.20.1)
