# 3.18 Releases

Convox 3.18 is the most feature-rich release line, with 12 patch releases over 5 months. Major additions include AWS RDS managed databases, ElastiCache Redis/Memcached, EFS volume support, NLB with Security Group, EKS Pod Identities, wildcard certificate reuse, and custom resource images.

## 3.18.0

**Released:** 2024-04-08

**Feature Additions**

- Added watch CLI option for list commands ([PR #758](https://github.com/convox/convox/pull/758))
- Added NLB with Security Group support ([PR #757](https://github.com/convox/convox/pull/757))

**Updates**

- Switched to aws-load-balancer-controller for NLB management ([PR #757](https://github.com/convox/convox/pull/757))

[View on GitHub](https://github.com/convox/convox/releases/tag/3.18.0)
