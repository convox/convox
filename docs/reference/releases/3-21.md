# 3.21 Releases

Convox 3.21 upgrades Kubernetes to 1.31 and introduces NVIDIA GPU device plugin support, additional node group tags and IDs, release-based environment variable management, and EBS volume encryption.

## 3.21.0

**Released:** 2025-04-02

**Updates**

- Upgraded Kubernetes to v1.31 ([PR #875](https://github.com/convox/convox/pull/875))
- Updated BuildKit to v0.20.1 ([PR #875](https://github.com/convox/convox/pull/875))
- Updated EBS CSI Driver to v1.41.0-eksbuild.1 ([PR #875](https://github.com/convox/convox/pull/875))
- Updated EFS CSI Driver to v2.1.6-eksbuild.1 ([PR #875](https://github.com/convox/convox/pull/875))
- Updated Kube Proxy to v1.31.2-eksbuild.3 ([PR #875](https://github.com/convox/convox/pull/875))

[View on GitHub](https://github.com/convox/convox/releases/tag/3.21.0)

## 3.21.1

**Released:** 2025-04-14

**Feature Additions**

- Added support for ID and custom tags in additional node group configuration ([PR #880](https://github.com/convox/convox/pull/880))
- Enabled NVIDIA GPU device plugin support via rack parameter ([PR #879](https://github.com/convox/convox/pull/879))
- Introduced release-based environment variable management ([PR #878](https://github.com/convox/convox/pull/878))

**Fixes**

- Initial fix to add readinessProbe for gRPC services ([PR #877](https://github.com/convox/convox/pull/877))

[View on GitHub](https://github.com/convox/convox/releases/tag/3.21.1)
