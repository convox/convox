# update

## CLI

Check CLI Version:

### Usage
```bash
    convox version
```
### Examples
```bash
    $ convox version
    client: 3.15.1
```

Update the CLI

### Usage
```bash
    convox update
```
### Examples
```bash
    $ convox update
    Updating to 3.16.0... OK
```

## Rack

For detailed information about updating racks, please visit our [Rack Management](https://docs.convox.com/management/cli-rack-management/) page.
