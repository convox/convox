# restart

The `convox restart` command performs a rolling restart of all services in an app. Each service is restarted sequentially to help maintain availability during the process.

## restart

Restart an app

### Usage
```bash
    convox restart
```
### Examples
```bash
    $ convox restart
    Restarting app... OK
```
